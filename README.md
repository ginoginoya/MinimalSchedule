# MinimalSchedule - 極簡桌面行程安排面板

這是一個基於 **Rainmeter** 開發的桌面行程管理皮膚，採用 **WebView2 (HTML/CSS/JS)** 技術實作，提供高質感的毛玻璃介面與流暢的動態收納功能。

## 🛠 必要需求
本皮膚必須具備以下組件才能正常執行：
1.  **Rainmeter**: 請確保已安裝最新版本的 [Rainmeter](https://www.rainmeter.net/)。
2.  **WebView2 Runtime**:請確保已安裝Rainmeter的外掛，請參考https://github.com/NSTechBytes/WebView2#-quick-start。
3.  **WebView2 Plugin (WebView2.dll)**: 
    *   本專案核心使用由 **NSTechBytes** 開發的 WebView2 外掛。
    *   **原作者 GitHub:** [NSTechBytes](https://github.com/NSTechBytes)

## ✨ 核心特色
*   **24小時排程管理：** 直觀的雙欄式任務清單。
*   **象限判定收納演算法：** 自動偵測面板在螢幕上的位置，智慧判斷收摺方向，解決移動不自然的問題。
*   **智慧吸附：** 展開或收合時自動吸附於螢幕邊緣或工作列 (Taskbar)。
*   **高度自定義：** 支援多種主題色（深海藍、翡翠綠等）與透明度即時調整。
*   **毛玻璃質感：** 現代化的 Glassmorphism 設計，美觀且不干擾桌面。

## 📝 使用說明與注意事項
1.  **安裝外掛：** 請確保將 `WebView2.dll` 放置於 Rainmeter 的 `Plugins` 目錄下（通常位於 `C:\Program Files\Rainmeter\Plugins`）。
2.  **首次載入：** 若載入後顯示空白，請對右鍵選單點擊 `Reload` 或檢查 WebView2 Runtime 是否安裝。
3.  **調整位置：** 
    *   面板支援滑鼠拖動（點擊頂部標題區域）。
    *   建議開啟「釘選置頂」功能，以獲得最佳的吸附效果。
